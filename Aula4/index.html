O que é o HTML?

O HTML é uma linguagem de marcação usada para estrutar páginas na web. Ele define titulos, parágrafos, imagens, links, etc.

O que é o CSS?

O CSS é uma linguagem usuda para estelizar páginas HTML. Ele controla cores, fontes, espaçamentos e o layout dos elementos.
